### [Style Guide](style-guide.md)

## Quick Reference

### Key Standards
- Follow EARS patterns for all requirements
- Use Mermaid for all diagrams
- Maintain requirements traceability
- Follow academic writing standards
- Use consistent formatting and style

### Quality Checklist
- [ ] Document follows template structure
- [ ] Academic writing standards met
- [ ] Requirements properly formatted
- [ ] Diagrams are professional and accurate
- [ ] Cross-references are valid
- [ ] Document information is complete

## Usage Guidelines

1. **Before Creating Documents**: Review relevant standards and templates
2. **During Creation**: Follow style guide and formatting standards
3. **Before Submission**: Complete quality checklist and review process
4. **After Updates**: Verify continued compliance with standards
